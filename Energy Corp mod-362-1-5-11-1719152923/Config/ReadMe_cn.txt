本文件用于描述Config文件夹内的文件和文件夹。
可使用快捷键Ctrl+F搜索特定的文件。

0. LcId 与 权重
LcId 由 '模组 ID + 异想体 ID'组成, 由','分隔。
每个LcId以换行符(Enter)分隔。
在安装模组后, 当你运行游戏时, Config文件夹内将生成一个叫做allCreatureList.txt的文件。
这个文件包含了几乎所有异想体，包括模组异想体的LcId。你可以在本文件中轻松地找到并复制异想体的LcId。

在权重节点中, 应有'LcId + 权重'.
“权重”应为一个介于0-1之间的浮点数，减小它就会降低出现概率，增大它就会提高出现概率。
当随机选取一个异想体时，如果它包含在权重节点中，它的权重就会转化为抽取异想体的概率。
如果它被选取了，这个异想体就会出现；否则，将从同一危险等级列表中随机抽取另一个异想体。
因此，即使一个异想体的权重为0，它也有小概率出现。

1. applymod.xml
应用或停用模块。

	1) addIsolate
	这是“额外的收容单元”模块。
	执行Legacy版本的额外收容单元。
	工具异想体不会被额外提取。
	数据类型: 布尔型, 默认值: true

	2) tt05
	这是“TT0.5”模块。你可以更改游戏速度的值。
	数据类型: 布尔型, 默认值: true

	3) energyRefine
	这是“能源精炼工作”模块。 执行Legacy版本的能源精炼工作。
	数据类型: 布尔型, 默认值: true

	4) isolateRandomize
	这是“收容单元重排”模块。
	从第七天开始，异想体的位置会在每天结束时随机重排。
	数据类型: 布尔型, 默认值: false

	5) overloadPlus
	这是“增加逆卡巴拉过载”模块。
	所有的非工具异想体都会受比以前数值更高的逆卡巴拉熔毁影响。
	数据类型: 布尔型, 默认值: true

	6) aberration
	这是“变体”模块。
	数据类型: 布尔型, 默认值: true

	7) centralStair
	这是“中央本部阶梯”模块。你可以借助透明的阶梯快速地穿过中央本部。
	数据类型: 布尔型, 默认值: false


2. modconfig.xml
你可以修改模组的细节。

	1) addIsolate
	这是“额外的收容单元”模块。它增加了你需要管理的异想体的数量。

		随机选取
		这是“随机额外异想体”模块。 无需你选择，额外的异想体就会被随机选取。
		数据类型: 布尔型, 默认值: false

	2) tt05
	这是“TT0.5”模块。你可以更改游戏速度的值。
	从上至下的指令依次为：暂停，正常速度，1.5倍速，2倍速。

		(1)暂停
		数据类型:浮点型, 范围:≥0, 默认值:0.5
		
		(2)1级速度
		数据类型:浮点型, 范围:≥0, 默认值:1
		
		(3)2级速度
		数据类型:浮点型, 范围:≥0, 默认值: 1.5
		
		(4)3级速度
		数据类型:浮点型, 范围:≥0, 默认值:2

	3) energyRefine
	这是“能源精炼工作”模块。

		(1) maxLobPenalty
		Maximum value of LOB Point Penalty.
		数据类型: 整形,范围:≥0, 默认值: 8

		(2) refineEfficiency
		数值越高，能源精炼工作完成的速度越快。
		数据类型:浮点型, 范围:≥0, 默认值: 15

		(3) ignoreBossFight
		如果在核心抑制的时候集满能源，跳过核心抑制并进入能源精炼工作阶段，
		此时视为核心抑制失败。
		数据类型: 布尔型, 默认值: false

	4) overloadPlus
	这是“增加逆卡巴拉过载”模块。每次你完成工作的时候，
	工作成功率会根据你给每个危险等级设定的值降低。
		(1) zayin
		数据类型: 整形, 范围: ≥0, 默认值: 4
	  
		(2) teth
		数据类型: 整形, 范围: ≥0, 默认值: 5
	  
		(3) he
		数据类型: 整形, 范围: ≥0, 默认值: 6
	  
		(4) waw
		数据类型: 整形, 范围: ≥0, 默认值: 7
	  
		(5) aleph
		数据类型: 整形, 范围: ≥0, 默认值: 8

	5) aberration
	这是“变体”模块。

		(1) selectable
		决定变体是否在异想体选择阶段出现。
		数据类型: 布尔型, 默认值: true

		(2) transformBlacklist
		O-05-47-1将不会变成列表中的异想体。
		数据类型: LcId, 默认值:
		100019
		100022
		100024
		Qquickness_EnergyCo, 178801
		Qquickness_EnergyCo, 178805

4. randomExtraction
独属于“随机额外异想体”模块的设置。

	1) blacklist.xml
	它能防止特定模组的异想体出现。

		(1) useBlacklist
		是否启用模组黑名单功能。
		数据类型: 布尔型, 默认值: false

		(2) blacklist
		你可以在这写下你想要屏蔽的模组的ID。
		如果你想要屏蔽多个模组，请用换行符(Enter)分隔模组ID。
		数据类型:字符串

	2) dailyDifficulty.xml
	调整“随机额外异想体”模块的难度。
	依次为Zayin、Teth、He、Waw、和Aleph。数字越大，它就更可能被选取。
	3) reserveCreature.xml
	你可以在你想要的日期以你想要的概率碰见你想要的异想体。
	创建一个"D + 想要的日期'节点。
	没有随机异想体的天数不会出现。
	4) risklevel.xml
	你可以通过添加或移除异想体的ID以增加或减少随机选取的概率。
	在节点中危险等级低的异想体更可能在游戏的开始出现，但随着游戏进行，它将更难出现。
	在节点中危险等级高的异想体在游戏的开始不会出现，但随着游戏进行，它将更可能出现。
		(1)创建一个risklevel.xml文件
		在riskLevel文件夹中，创建一个名字包含“risklevel”、格式为xml的文件，来让本模组识别它。
		请参考一个已存在的risklevel.xml的文件结构。它应该包含一个阻止随机选取节点和一个权重节点。
