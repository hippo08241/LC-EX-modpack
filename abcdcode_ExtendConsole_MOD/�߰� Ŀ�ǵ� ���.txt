extend byebyeego [직원번호] [0이면 무기, 1이면 방어구]
: 해당 직원이 지닌 장비를 파괴한다. 직원 번호를 0으로 하면 모든 직원에게 일괄 적용

extend youfired [직원번호]
: 해당 직원을 해고한다

extend despell [직원번호]
: 해당 직원에게 부여된 모든 버프를 해제한다. 직원번호를 0으로 하면 모든 직원에게 일괄 적용

extend addstat [직원번호] [바꿀 스탯] [수치]
: 해당 직원의 특정 스탯을 수치만큼 변화시킨다. 직원번호를 0으로 하면 모든 직원에게 일괄 적용
바꿀 스탯
- 0 : 용기
- 1 : 지혜
- 2 : 절제
- 3 : 정의
- 4 : 모든 능력치

extend setstat [직원번호] [바꿀 스탯] [수치]
: 해당 직원의 특정 스탯을 입력한 수치로 바꾼다. 직원번호를 0으로 하면 모든 직원에게 일괄 적용

extend killing_monster [환상체 번호]
: 해당 번호의 환상체를 즉시 제압한다. 번호를 0으로 하면 모든 환상체를 제압한다.

extend homework
: 모든 사무직을 퇴근시킨다. 사망이 아닌 삭제 판정이므로 큰새, 웃는 시체들의 산 등의 관리 조건에 영향을 미치지 않는다.
